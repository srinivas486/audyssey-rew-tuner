## Story 2.1 — Write REW .frd format per channel

**File:** rew_exporter.py (create new)

```python
def export_channel_frd(freq_hz, spl_db, output_path, channel_name) -> bool
```

Requirements:
- Write a REW-compatible .frd file with one freq_hz spl_db pair per line, space or comma-delimited
- Frequency must be in ascending order (required by REW)
- Output path: {output_path}/{channel_name}.frd
- REW .frd comment rows (optional, starts with # or *) are fine but not required
- Returns True on success, False on error
- freq_hz and spl_db are np.ndarray from ady_parser.get_channel_freq_response() averaged data

Acceptance Criteria:
- export_channel_frd() writes valid .frd with ascending freq, correct spl_db values
- File written to expected output path
- Works with np.ndarray input (converts to sorted list internally)

## Story 2.2 — Push via REW API with graceful fallback

**File:** rew_exporter.py (extend with second function)

```python
def push_frequency_response_via_api(freq_hz, spl_db, channel_name, host="localhost", port=4735) -> bool
```

Requirements:
- POST JSON to http://{host}:{port}/import/frequency-response-data
- Body format: {"FrequencyResponseData": {"frequency": [...], "level": [...]}}
- Arrays must be plain Python lists (not numpy arrays) — convert with .tolist()
- frequency in Hz, level in dB (already dB from FFT extraction in ady_parser)
- Returns True if REW accepts data (HTTP 200/202), False if API unavailable
- Graceful fallback: catch ConnectionError/httpx.ConnectError, print warning "REW API not available at localhost:4735 — skipping", return False
- No crash if REW not running
- Use urllib.request (standard library, consistent with rew_to_audyssey.py load_from_rew pattern)

Acceptance Criteria:
- push_frequency_response_via_api() POSTs correct JSON to REW API
- Returns False gracefully when REW is not running (no exception crash)
- Prints warning when REW unavailable: "REW API not available at localhost:4735 — skipping"

## Integration pattern

```python
from ady_parser import load_ady, get_all_channels_freq_response
from rew_exporter import export_channel_frd, push_frequency_response_via_api

data = load_ady("measurement.ady")
channel_responses = get_all_channels_freq_response(data)

for ch in channel_responses:
    cmd_id = ch["commandId"]
    avg = ch["averaged"]
    freq = avg["freq_hz"]       # np.ndarray
    spl  = avg["spl_db"]         # np.ndarray

    export_channel_frd(freq, spl, "./output", cmd_id)
    push_frequency_response_via_api(freq, spl, cmd_id)
```
